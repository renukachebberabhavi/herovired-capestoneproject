Problem Statement:
Create a Job portal website where:
A candidate will:

Create account
Email
Password
Login
Add details in profile like
Phone number
Current company
Current CTC
Current role
About
Profile picture
Upload CV
Add skills
View the list of jobs
Refer Figma design for the information
Apply in those jobs
